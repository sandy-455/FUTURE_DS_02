# FUTURE_DS_02
📊 Marketing Funnel & Conversion Analysis – Task 2

🔍 Project Overview

This project focuses on analyzing user behavior in an e-commerce environment to understand how users move through the marketing funnel (View → Cart → Purchase).

The goal is to identify drop-off points, measure conversion rates, and provide data-driven business recommendations to improve overall performance.

⸻

🎯 Objectives
	•	Analyze user journey across funnel stages
	•	Calculate conversion rates at each stage
	•	Identify where users drop off
	•	Understand customer behavior patterns
	•	Provide actionable recommendations to improve conversions

⸻

📂 Dataset

The dataset contains user interaction data with the following key features:
	•	event_time – Timestamp of user activity
	•	event_type – Type of event (view, cart, purchase)
	•	product_id – Product identifier
	•	category_code – Product category
	•	brand – Product brand
	•	price – Product price
	•	user_id – Unique user identifier
	•	user_session – Session identifier

📈 Key Insights
	•	A large number of users drop off at the View → Cart stage
	•	Users who add items to cart have a higher chance of purchasing
	•	Certain product categories and brands perform better in conversions
	•	Price and timing influence purchase behavior
